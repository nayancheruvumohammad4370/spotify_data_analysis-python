# Spotify Data Analysis

# Project Overview
This project focuses on analyzing Spotify music data to discover insights about songs, artists, popularity, and audio features using Python.

# Dataset Description
The dataset contains Spotify track-level information such as track name, artist, popularity score, duration, and various audio features.

# Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn

# Jupyter Notebook
Data Types
Textual (Categorical): Track name, artist name, genre
Numerical: Popularity, duration, tempo, loudness
Dimensional: Audio feature-based attributes used for analysis
Analysis Performed
Data cleaning and preprocessing
Handling missing values
Exploratory Data Analysis (EDA)
Statistical and trend analysis

# Data visualization
Key Insights
Popularity varies significantly across artists and genres
Certain audio features influence song popularity
Data trends reveal listener preferences

# Visualizations
